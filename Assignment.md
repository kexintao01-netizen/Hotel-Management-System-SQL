# Hotel Management System Database Design and SQL Analysis

## Project Background

A hotel needs a database system to manage daily operations, including guest information, room management, room bookings, payments, staff, departments, hotel services and customer reviews.

This project requires students to design and implement a relational database for a hotel management system. Students need to create tables, define relationships, insert sample data and write SQL queries to answer practical business questions.

## Project Requirements

Students are required to:

1. Design a relational database for a hotel management system.
2. Create tables using SQL.
3. Define primary keys and foreign keys.
4. Insert sample data into each table.
5. Write SQL queries to answer business-related questions.
6. Demonstrate SQL techniques including joins, aggregation, subqueries, CASE WHEN, CTEs, views and window functions.

## Suggested Tables

| Table | Description |
|------|-------------|
| Guest | Stores guest information |
| RoomType | Stores room type and price information |
| Room | Stores individual room information |
| Booking | Stores booking records |
| BookingRoom | Links bookings with rooms |
| Payment | Stores payment records |
| Department | Stores hotel departments |
| Staff | Stores staff information |
| Service | Stores hotel service information |
| ServiceOrder | Stores guest service orders |
| Review | Stores customer reviews |

## SQL Query Questions

1. List all guests and their booking information.
2. Show all rooms with their room type, price per night and current status.
3. Find the total revenue generated from each room type.
4. Find guests who have spent more than £500 in total.
5. Classify guests based on their total spending.
6. Find the most frequently booked room.
7. Find guests whose spending is higher than average guest spending.
8. Rank guests by their total spending.
9. Find the average review rating for each room type.
10. Create a view for guest booking and payment information.

## Expected Outcome

After completing this project, students should be able to demonstrate the ability to design a relational database and use SQL to retrieve, analyse and summarise hotel management data.
